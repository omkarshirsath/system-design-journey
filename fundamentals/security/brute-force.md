# Brute Force Attack

## What Is a Brute Force Attack?

Brute force is an attack where an attacker systematically tries many
password combinations until the correct one is found.

---

## Real World Example

Automated script tries:

admin123
admin1234
password
Password1!
password123

Thousands of attempts per second.

---

## Impact

- Account compromise
- Credential stuffing
- Lateral movement
- Data breach

---

## Prevention

- Rate limiting per IP/user
- Account lockout after failures
- CAPTCHA challenges
- MFA requirement

---

## Common Technologies Affected

- Login endpoints
- API authentication
- SSH/RDP services
- Admin panels

---

## Modern Protection Techniques

- Progressive delays
- Bot detection
- Allowlisting/blocklisting
