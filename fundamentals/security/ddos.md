# Distributed Denial of Service (DDoS)

## What Is DDoS?

Distributed Denial of Service is an attack where thousands of infected
machines (botnet) simultaneously flood a target server with traffic.

---

## Real World Example

A botnet of 100,000 compromised IoT devices sends HTTP floods to
`https://example.com`, saturating the network link.

---

## Impact

- Complete network saturation
- Massive downtime
- Infrastructure collapse
- ISP-level blackholing

---

## Prevention

- CDN (Cloudflare, Akamai)
- Anycast routing
- Rate limiting at edge
- Traffic scrubbing centers

---

## Common Technologies Affected

- DNS servers
- Origin servers
- Load balancers
- Network infrastructure

---

## Modern Protection Techniques

- DDoS mitigation services
- Web Application Firewall (WAF)
- Behavioral analysis
